# README



Things you need want to cover:

* Ruby version: 2.6.3
* Rails version: Rails 5
* Postgresql

* Configuration
Run
````
bundle install
````

* Database creation : 
Run
````
rake db:create 
rake db:migrate 
````
OR
````
rails db:create
rails db:migrate
````

* Deployment instructions
Run
````
rails s
````
